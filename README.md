# Node TypeScript Template

## To Run

```bash
## build script
npm run build

## Developer mode
npm run dev

## Production run
npm start

```

## Git Clone

```bash
## Cloning main Repo
git clone https://github.com/Arkaraj/Node-Ts_Template.git

## Cloning template with mongodb, mongoose passport-jwt authentication
git clone -b mongodb_passport-auth --single-branch https://github.com/Arkaraj/Node-Ts_Template.git

## Cloning template with mysqldb typeorm passport-jwt authenticaiton
git clone -b mysql_passport-auth --single-branch https://github.com/Arkaraj/Node-Ts_Template.git

```
